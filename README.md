# 🎉 Event Management Dashboard

This is a full-featured Event Management Dashboard built using **Next.js (React + TypeScript)** with:

- 🔐 Local authentication (Sign up / Login)
- 🗓️ Event CRUD (Create, Read, Update, Delete)
- 📅 Date conflict validation
- 🔍 Filtering, Sorting, Searching
- ⚙️ JSON Server used as mock backend

---

## 🧾 Features

- LocalStorage-based user authentication
- Persistent login sessions
- Add / Edit / Delete events
- Prevent overlapping event times
- Search, filter by event type, category
- Sort by title or date
- Query params persisted in URL
- React Hook Form-based validation

---

## 🛠️ Tech Stack

- Next.js (App Router)
- TypeScript
- React Hook Form
- React Context API
- Tailwind CSS
- JSON Server (Fake REST API)
